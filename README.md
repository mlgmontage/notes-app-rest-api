## Simple & basic note app REST API

Simple & Powerful REST API for note taking app.

## Tech-Stack

- express
- prisma-client
- dotenv
- volleyball
- cors
- jsonwebtoken
- joi
- bcrypt

## API

### Notes

- `GET: /api/notes` - list of notes
- `POST: /api/notes/create` - create note

### Users

- `POST: /api/users/signup` - register
- `POST: /api/users/signin` - login
